run

An run command lets you run an other test from this test. The purpose is to support suites better.

Example

The following instrumentation:

<p>
<a concordion:run="testrunner" href="RunTrue.html">Will execute RunTrue.html with true result</a>
</p>

We get the following outcomes depending on the result of the RunTestRunner:

Expected Href Run Result Outcome
RunTrue.html true SUCCESS
NotExisting.html false FAILURE
